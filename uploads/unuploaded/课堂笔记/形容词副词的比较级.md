



| 变化规则类型                                              | 原级示例        | 比较级形式          | 最高级形式           |
| --------------------------------------------------- | ----------- | -------------- | --------------- |
| 一般情况：词尾直接加 _-er_ / _-est_                           | small       | smaller        | smallest        |
|                                                     | tall        | taller         | tallest         |
| 以**不发音的 _e_** 结尾：加 _-r_ / _-st_                     | large       | larger         | largest         |
|                                                     | nice        | nicer          | nicest          |
| 重读闭音节（结构：**辅 + 元 + 辅**）：双写末尾辅音，再加 _-er_ / _-est_    | big         | bigger         | biggest         |
|                                                     | hot         | hotter         | hottest         |
| 以 **“辅音字母 + _y_”** 结尾：变 _y_ 为 _i_，再加 _-er_ / _-est_ | easy        | easier         | easiest         |
|                                                     | happy       | happier        | happiest        |
| 部分**双音节词** + 所有**多音节词**：前加 _more_ / _most_          | beautiful   | more beautiful | most beautiful  |
|                                                     | important   | more important | most important  |
|                                                     | quickly（副词） | more quickly   | most quickly    |
| 前加 _less_ / _least_                                 | expencive   | less expencive | least expencive |






不规则

| 原级（形容词 / 副词）                                                   | 比较级形式                                              | 最高级形式                                                 | 用法备注                         |
| -------------------------------------------------------------- | -------------------------------------------------- | ----------------------------------------------------- | ---------------------------- |
| _good_（形，“好的”）<br><br>_well_（副，“好地”）                           | _better_                                           | _best_                                                | _well_ 仅作副词时，变化同 _good_      |
| _bad_（形，“坏的”）<br><br>_ill_（形，“生病的”）<br><br>_badly_（副，“糟糕地”）    | _worse_                                            | _worst_                                               | _ill_ 表 “生病”；_badly_ 表 “程度糟” |
| _many_（形，“许多”，修饰可数名词）<br><br>_much_（形，“许多”，修饰不可数名词；副，“很 / 非常”） | _more_                                             | _most_                                                | _many/much_ 共享比较级 / 最高级      |
| _little_（形 / 副，“少的 / 少地”）                                      | _less_                                             | _least_                                               | 表 “数量少” 或 “程度轻”              |
| _old_（形，“旧的 / 年龄大的”）                                           | _older_（表 “更旧；年龄更大”）<br><br>_elder_（表 “长幼关系里更年长”）  | _oldest_（表 “最旧；年龄最大”）<br><br>_eldest_（表 “长幼关系里最年长”）   | 双重变化，依 “描述对象” 选用法            |
| _far_（形 / 副，“远的 / 远地”）                                         | _farther_（表 “距离更远”）<br><br>_further_（表 “程度更深；进一步”） | _farthest_（表 “距离最远”）<br><br>_furthest_（表 “程度最深；最进一步”） | 双重变化，依 “描述维度” 选用法            |
