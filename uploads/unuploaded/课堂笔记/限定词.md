
| 类别       | 说明                                                  | 示例                                                               |
| -------- | --------------------------------------------------- | ---------------------------------------------------------------- |
| 冠词       | 分不定冠词（_a_/_an_，表 “一个 / 某一”）和定冠词（_the_，表 “特指”）       | _a dog_（一只狗）；_the dog_（这只狗）                                      |
| 形容词性物主代词 | 表所属关系，功能相当于形容词，修饰名词                                 | _her bag_（她的包）；_our classroom_（我们的教室）                            |
| 名词所有格    | 表 “…… 的”，分 _’s_ 所有格（多表有生命事物所属）和 _of_ 所有格（多表无生命事物所属） | _Tom’s bike_（汤姆的自行车）；_the leg of the table_（桌子的腿）                |
| 指示代词     | 指示 / 标识人或事物，分近指、远指，有单复数                             | _that house_（那座房子）；_these flowers_（这些花）                          |
| 不定代词     | 不明确指代特定人 / 事物，部分需区分可数 / 不可数、肯定 / 否定表意               | _some water_（一些水）；_each child_（每个孩子）                             |
| 基数词      | 表数量的数词，修饰可数名词复数                                     | _two pens_（两支钢笔）；_five days_（五天）                                 |
| 疑问代词     | 引导特殊疑问句，在句中可作主语、宾语、表语等，指人或物                         | _What book do you want?_（你想要什么书？）；_Whose coat is this?_（这是谁的外套？） |
