### 等价无穷小

| 等价关系                                       |
| ------------------------------------------ |
| $(sin x \sim x)$                           |
| $(tan x \sim x)$                           |
| $(\arcsin x \sim x)$                       |
| $(\arctan x \sim x)$                       |
| $(1 - \cos x \sim \frac{1}{2}x^2)$         |
| $(\ln(1 + x) \sim x)$                      |
| $(e^x - 1 \sim x)$                         |
| $(a^x - 1 \sim x\ln a)（(a > 0, a \neq 1)）$ |
| $((1 + x)^\alpha - 1 \sim \alpha x)$       |
